## hostname-deploy-svc-ns.yml
```yml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: hostname-deployment-ns
  namespace: production
spec:
  replicas: 3
  selector:
    matchLabels:
      app: webserver
  template:
    metadata:
      name: my-webserver
      labels:
        app: webserver
    spec:
      containers:
      - name: my-webserver
        image: alicek106/rr-test:echo-hostname
        ports:
        - containerPort: 80
---
apiVersion: v1
kind: Service
metadata:
  name: hostname-svc-clusterip-ns
  namespace: production
spec:
  selector:
    app: webserver
  ports:
  - name: web-port
    port: 8080
    targetPort: 80
  type: ClusterIP
```

## all-env-from-configmap.yml
```yml
apiVersion: v1
kind: Pod
metadata:
  name: container-env-example
spec:
  containers:
  - name: my-container
    image: busybox
    args: [ 'tail', '-f', '/dev/null' ]
    envFrom:
      - configMapRef:
          name: log-level-configmap
      - configMapRef:
          name: start-k8s
```
---

## selective-env-from-configmap.yml
```yml
apiVersion: v1
kind: Pod
metadata:
  name: container-env-example2
spec:
  containers:
  - name: my-container2
    image: busybox
    args: [ 'tail', '-f', '/dev/null' ]
    env:
    - name: ENV_KEYNAME_1
      valueFrom:
        configMapKeyRef:
          name: log-level-configmap
          key: LOG_LEVEL
    - name: ENV_KEYNAME_2
      valueFrom:
        configMapKeyRef:
          name: start-k8s
          key: k8s
```
---

## volume-mount-configmap.yml
```yml
apiVersion: v1
kind: Pod
metadata:
  name: configmap-volume-pod
spec:
  containers:
  - name: my-container
    image: busybox
    args: [ 'tail', '-f', '/dev/null' ]
    volumeMounts:
      - name: configmap-volume
        mountPath: /etc/config
  volumes:
  - name: configmap-volume
    configMap:
      name: start-k8s
```
---
